Kestrel 5400 — Ordered How-To Steps

A. WBGT setup & alerts

  1. From the WBGT Current Measurement screen, press Select.
  2. Set Type = Indoor or Outdoor.
  3. Open Zones… and set to On to enable colored warning zones (White→Black by risk severity).
  4. To use a pre-saved organizational guideline, go to Zones… ▸ Guide… and choose the appropriate guide. If prompted, set Workload and Clothing.
  5. (Optional) For custom thresholds: in Zones…, edit a color zone’s Status (On/Off) and Threshold. Thresholds must remain ordered relative to adjacent zones.
  6. Configure alerts: Alerts… → set Light and/or Buzzer for the desired zone(s). Use Display ▸ Alerts Test to check intensity.
  7. During use: entering the Red zone causes the zone line to flash; entering the Black zone inverts the measurement. Press the power button on the WBGT screen to acknowledge an active alert; it will re-arm on the next threshold crossing.

B. Accurate WBGT measurement protocol

  • After changing environments, allow 8–15 minutes for sensors to equilibrate before relying on readings.
  • Measure about 3 ft (≈1 m) above ground and in the same airflow and solar/radiant environment as the people being monitored.
  • Orient the back of the unit into the wind. A tripod or pole with a rotating vane mount helps ensure full wind exposure.
  • Ground surface reflectivity (e.g., grass vs. asphalt) affects readings—match your team’s surface and sun/shade.

C. Device options & logging

  1. Open Options (gear icon).
  2. Memory Options:
    • Auto Store = On.
    • Store Rate = desired log interval (choose to suit your protocol).
    • Overwrite = On to wrap when full, or Off to stop logging when full.
  3. Graph Scale: choose a measurement, then set High/Low bounds for the display.
  4. Display ▸ Auto Shtdwn: pick an auto-shutdown time window.
  5. System ▸ Compass Cal: calibrate (flat surface, away from metal; rotate vertically 3 full turns, ~10 s each).
  6. Display ▸ Backlight: choose White or Red.
  7. System ▸ Time & Date: set clock.
  8. System ▸ Measurements: turn measurement screens On/Off as needed.
  9. System ▸ Units: set Global (Imperial/Metric) or per-measurement units.
  10. Lang: select language.

D. Logging & viewing data on the device

  • Manual capture: press Capture to add a time-stamped point.
  • Graph Data: from a measurement, go to the Graph screen to see recent logs; press Select for Data Log Detail and scroll to inspect points with timestamps.
  • Graph resolution: adjust via Options ▸ Graph Scale.

E. Exporting logs

  1. Export to a mobile device: install the Kestrel LiNK app (App Store/Play Store) and follow in-app instructions to export.
  2. Export to a computer (two methods):
    1. Wireless (LiNK dongle): plug in the USB LiNK dongle → pair via Kestrel LiNK → in LiNK, choose a log and Export to File.
    2. USB Data Transfer Cable: Options ▸ Data Port = On → connect the cable to the meter and computer → in LiNK, confirm connection → select log → Export to File.
