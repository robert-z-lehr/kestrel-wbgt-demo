Intersection — Crosswalk / Median Dwell

Objective

Compare conditions experienced while waiting at crosswalks/medians versus adjacent shaded areas.

Site Metadata
Intersection________
ControlSignalized / Stop / Other
Median typeBare / Planted / With shelter
Traffic intensityLow / Medium / High (notes)
Design
  • Paired spots: dwell area (sun) vs shade control.
  • Alternate measurements in matched time blocks.
  • Keep height/orientation identical across spots.
Procedure
  1. Stabilize; start in sun location (Capture mark).
  2. Log N intervals; note traffic pulses or bus arrivals.
  3. Move to shade control; stabilize; log N intervals.
  4. Repeat sequence if time permits.
Expected Outputs
  • Delta WBGT (sun − shade).
  • Component contributions: ΔTg (radiation), Δv (wind), ΔRH.
  • Photos annotated with positions.

Surface Comparison — Asphalt vs Grass

Objective

Quantify how surface type influences WBGT and components under similar meteorological conditions.

Design & Procedure
  • Two adjacent sites at same height/exposure: asphalt and grass.
  • Alternate in 10–15 min blocks; stabilize after each move.
  • Use Capture at each move and any cloud cover change.
Key Metrics
  • ΔWBGT, ΔTg, ΔTa; note wind differences.
  • Photograph surfaces and sky conditions.

Intervention — Shelter / Shade / Mister

Objective

Estimate change in WBGT and components after adding or using an intervention (e.g., bus shelter, shade sail, mister).

Procedure
  1. Baseline measurement (no intervention active).
  2. Activate intervention; wait to stabilize; measure again.
  3. Repeat if feasible to average out transient effects.

Wind Exposure — Orientation Sensitivity

Objective

Check the effect of orientation and shielding on readings, ensuring placement reflects people’s exposure.

Procedure
  • Compare open exposure vs. near-wall/vehicle shielding.
  • Log at least N intervals in each; maintain height.
  • Note gusts; use Capture during obvious wind shifts.

External Validation — Reference Comparison

Objective

Compare your readings to a nearby reference (e.g., mesonet station, airport METAR) acknowledging siting differences.

Procedure
  • Match time windows; record reference data source and timestamps.
  • Compare Ta, RH, wind; discuss reasons for expected differences.
